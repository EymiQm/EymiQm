
# Mars Battery Monitoring System

## Description
This project is a simple Python program that checks the battery level after daily energy usage.  
It is a basic algorithm designed to simulate a monitoring system for survival equipment.

## How it works

1. The program defines two variables: battery and daily_use.
2. The function check_battery calculates the remaining battery.
3. The program subtracts daily energy use from the total battery.
4. The system checks if the remaining battery is greater than twenty.
5. If the battery is safe, the program returns a safe message.
6. If the battery is low, the program returns a warning message.
7. Finally, the program prints the result.

## Status

> The project is currently running as a basic simulation of a battery monitoring system.
